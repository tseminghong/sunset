An interactive guide to system and application software.

A. System Software

1. Operating System (OS)

An Operating System (OS) is crucial system software that manages computer hardware and software resources and provides common services for computer programs.

Examples: Windows, MacOS, Android, iOS, iPadOS

For desktop computers For mobile devices For servers

Common Functions/Features:

Job scheduling and running

- Handles multitasking jobs efficiently.

- Provides services for computer programs to execute.

Manage system resources

Controls and allocates resources like:

  • CPU time
  • Memory (RAM)
  • Virtual memory
  • Storage (disk space)
  • Network usage

Error handling / Show alerts

- Detects and manages system errors.

- Shows alerts to the user (e.g., "not enough storage/memory" alerts).

- Manages system security and error detection.

Provision of user interface

Allows users to interact with the system through various interface types:

Graphical User Interface (GUI)

Visually intuitive interface using icons, windows, and menus.

Pros: Easy control for beginners, visually appealing.

Cons: Consumes more computer resources.

Examples: Windows, macOS

Menu-driven interface

Users interact by selecting options from a series of menus.

Pros: No need to remember complex commands, easier for new users than CLI.

Cons: Can be slower for experienced users, navigation can be tedious for complex tasks.

Command-line interface (CLI)

Text-based interface where users type commands to interact with the system.

Pros: Fast control for advanced users, consumes very little computer resources, powerful for automation.

Cons: Steep learning curve, requires knowledge of commands, less intuitive.

Examples: Terminal (Linux/macOS), Command Prompt (Windows)

Maintain security

- Manages different user permissions and access levels.

- Protects user privacy and data integrity.

Control and management of hardware

- Manages and communicates with hardware devices.

- Makes use of driver programs to interact with specific hardware components.

Network management

- Monitors network traffic and connections.

- Manages network configurations and protocols.

File management

- Helps users explore files in storage.

- Manages file creation, deletion, modification, and access to files.

Provide command line interface

- Offers a text-based interface for advanced users to interact with the system directly (see comparison above).

Provide a platform for application programs

- Creates an environment where application software can run and access system resources.

Mobile O.S. specific features

User interface responding to touch gestures:

  • Swipe
  • Scroll
  • Click (tap)
  • Pinch to zoom
  • Long press
  • Double click (double tap)
Operating System Summary
  • Controls hardware
  • Provides user interface
  • Manages resources (CPU, memory, storage, etc.)

2. Utility Program

Utility programs perform specific tasks related to managing and maintaining the computer system. They assist the OS in optimizing performance and security.

  • Antivirus program: Detects and removes malicious software (viruses, malware).
  • Defragmenter / File defragmenter: Recombines fragmented files on a storage disk to improve file access speed.
  • Software firewall: Monitors and controls incoming and outgoing network traffic to protect against unauthorized access.
  • Storage cleaner: Helps free up disk space by removing unnecessary files (e.g., temporary files, old downloads).

3. Driver Program

A driver program (or device driver) is a special type of software that allows the O.S. to control the hardware devices.

  • Function: Acts as a translator between the OS and the hardware.
  • Pre-installed drivers: Many common devices (e.g., mouse, keyboard) have standard drivers pre-installed or automatically installed by the OS (Plug-and-Play).
  • Specific drivers: Some hardware may require specific driver programs to be installed manually by the user to enable all its features or for optimal performance (e.g., high-end graphics cards, specialized printers).

B. Application Software

Application software consists of programs designed to perform specific tasks for end-users. Unlike system software, which manages the computer itself, application software is what users interact with directly to accomplish various activities.

Communication Software

  • Social media applications (e.g., Facebook, Instagram, X)
  • Web browsers (e.g., Chrome, Firefox, Safari)
  • AI chatbot software (e.g., ChatGPT, Google Gemini)
  • Email clients (e.g., Outlook, Gmail)
  • Messaging apps (e.g., WhatsApp, Telegram)

Entertainment Software / Gaming Software

  • Video games (PC, console, mobile)
  • Streaming services (e.g., Netflix, Spotify)

Media Player

  • Software for playing audio and video files (e.g., VLC Media Player, Windows Media Player).

Media Editing Programs

  • Photo editor: (e.g., Adobe Photoshop, GIMP) - For manipulating digital images.
  • Video editor: (e.g., Adobe Premiere Pro, DaVinci Resolve) - For creating and modifying video content.
  • Audio editor: (e.g., Audacity, Adobe Audition) - For recording and editing sound.
